## 📚 Descripción

Este repositorio contiene material educativo sobre el **Método Montecarlo**, una técnica de simulación estadística utilizada para resolver problemas determinísticos y estocásticos mediante el uso de números aleatorios.

## 📂 Contenido

| Archivo | Descripción |
|---------|-------------|
| [docs/montecarlo-metodo.md](docs/montecarlo-metodo.md) | Explicación detallada del método Montecarlo |
| [docs/ejemplo-pi.md](docs/ejemplo-pi.md) | Cálculo de π usando el método Montecarlo |
| [docs/ejemplo-integral.md](docs/ejemplo-integral.md) | Aproximación de integrales definidas |

## 🎯 Objetivos

- Comprender los fundamentos del método Montecarlo
- Aprender a generar números pseudoaleatorios
- Aplicar el método a problemas prácticos
- Visualizar los resultados mediante gráficas

## 🛠️ Requisitos

- Python 3.8+
- Librerías: numpy, matplotlib, scipy
